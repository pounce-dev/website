Prerequisites

A list of software and knowledge to participate in the following lessons.

Knowledge

A knowledge of binary, hexadecimal, the operating system you are using and how to install software/run commands are all necessary in order to understand what we are doing.

Software

Install the following software to participate in the lessons (Latest/stable Builds should always work):

QEMU

Download here.
Windows: Download the installer from Stefan Weil install QEMU. Once it is installed, add the path to your qemu folder (usually C:\Program Files\qemu\) into your system's environmental variables PATH.
Linux: Install with your package manager.

Check to see if it is working by running:

qemu-system-x86_64 --version

NASM

Download here.
Windows: Use the installer provided in the stable build.
Linux: You may install with your package manager.

Check to see if it is working by running:

nasm --version
